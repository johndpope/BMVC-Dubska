diamond_space_results.mat

	Results of diamond space detection on York Urban Database.
	MATLAB structure with fields:
	DiamondSpaceRes(i).Image - image name
	DiamondSpaceRes(i).Detected - 3x3 matrix with three detected vanishing points, each in one column
	DiamondSpaceRes(i).Orthogonal - 3x3 matrix with three vanishing points after orthogonalization, each in one column
